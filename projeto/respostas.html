Questionário

  1. Os termos Web e Internet frequentemente são usados como sinônimos. Explique com suas palavras qual é a diferença entre Internet e Web?

    2. A diferença é que: A internet, é uma rede que interliga vários computadores, Web é uma das ferramentas utilizadas para interpretar a comunicação entre estes computadores.

  2. O que acontece quando digitamos um endereço web na barra de navegação? Explique de maneira sucinta o funcionamento do modelo cliente/servidor da web. Para enriquecer sua resposta, adicione em sua explicação imagens estáticas ou gifs animados, caso encontre.

    3. Quando digitamos um um endereço web na barra de navegação, estamos indormando o endereço do site que desejamos acessar. O DNS por sua vez, converte o endereço informado para o endereço de IP do servidor e direciona os tipos de serviços desejados para as portas disponibilizadas. após essa interpretação, há uma solicitação por parte do cliente para acessar o endereço solicitado. Caso autorizado pelo servidor, inicia-se a transferencia de pacotes.
